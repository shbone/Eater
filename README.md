## 瑞吉外卖项目说明

本项目是SpringBoot+MybatisPlus+Mysql技术栈的前后端分离外卖管理系统。

本项目（瑞吉外卖）是专门为餐饮企业（餐厅、饭店）定制的一款软件产品，
包括 系统管理后台 和 移动端应用 两部分。 其中系统管理后台主要提供给餐饮企业内部员工使用，
可以对餐厅的分类、菜品、套餐、订单、员工等进行管理维护。 移动端应用主要提供给消费者使用，
可以在线浏览菜品、添加购物车、下单等。
